## Project Overview

Nuxt Auth Bridge bridges the gap between the powerful Better-Auth library and Nuxt 3. It provides auto-imported composables, middleware, and server-side utilities to handle authentication flows with ease.

## Key Features

- **Middleware Support:** Ready-to-use route guards for client and server.
- **Session Management:** Reactive session state with persistence.
- **Social Login:** Pre-configured providers for GitHub, Google, and more.

## Why it matters

Authentication is often a bottleneck in development. This bridge simplifies the process, allowing developers to focus on their core features without compromising on security.
